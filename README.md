**Telecom Customer Churn Prediction**

A simple machine learning project to analyze telecom customer data and predict whether a customer will churn (leave the service).

**Overview**

Loads telecom churn dataset

Performs data cleaning and EDA

Checks missing values, duplicates, and churn distribution

Builds ML models to predict customer churn

Evaluates performance using accuracy and classification metrics

**Tech Used**

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn, Jupyter Notebook

**How to Run**
git clone <repo-link>
pip install -r requirements.txt
jupyter notebook Telecom_Customer_Churn.ipynb
